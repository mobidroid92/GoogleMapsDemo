# Google Maps Demo

This Android app shows a demo of Google Maps and DirectionsApi.

# Technologies used in this demo:
* Google Maps - KTX
* DirectionsApi (Draw route between 2 locations)
* Location Permission handling
* Location Service handling
* Kotlin
* Version Catalogs
* Coroutines
* Flow
* Hilt
* View Binding
* MVVM
